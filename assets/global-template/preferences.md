# Personal Global Preferences (L1)

Your cross-project defaults and taste. Keep this SMALL. Overrides core; projects override this.

## Communication
- Terse, direct. Lead with the answer. Show diffs and file paths.

## Defaults (used only when a project doesn't specify)
- Defer to the project's configured formatter/linter; never hand-format.
- Prefer fast, deterministic, behavior-focused tests.
- Commits: Conventional Commits (feat/fix/refactor/test/docs/chore).

## Always want
- A one-line risk note after any non-trivial change.
- Proactive flagging of TODOs/hacks to review manually.

## Never want
- Speculative abstractions I didn't ask for.
- Reformatting unrelated files. Silent new dependencies.
